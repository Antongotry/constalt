# Constalt Workflow Rules

Эти правила считаются базовыми для всех следующих итераций работы.

## Основной подход

- Работаем в репозитории: `https://github.com/Antongotry/constalt`.
- Коммит и пуш делаем в этот репозиторий.
- Вы передаете фото и/или код, а новые блоки собираются по ним.
- Приоритет: переиспользуемые компоненты и `template-parts` (partial), чтобы повторяющиеся элементы не дублировались.

## Текущее состояние

- Главная страница и формы уже готовы.
- В проекте есть логика с двумя точками перехода.
- Следующий крупный этап: раздел/блоки блога.

## Правила адаптива и размеров

- Desktop: перевод размеров в `vw` относительно макета `1440`.
- Mobile: перевод размеров в `vw` относительно макета `375`.
- Технические значения (например, `1px` бордеры) можно оставлять в `px`.

## Стандарты разработки

- Новую секцию сначала проверяем, можно ли собрать из существующих partial.
- Если повторяется паттерн, выносим его в отдельный partial/component.
- Стили держим в SCSS partial-файлах по назначению секций.

## Напоминание для будущих итераций

Перед началом каждой новой задачи сверяться с этим файлом и соблюдать правила из него.
